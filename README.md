# 💬 Talk-A-Tive

**Talk-A-Tive** is a full-stack real-time chat application built using the **MERN stack** — MongoDB, Express.js, React.js, and Node.js — with **Socket.io** for instant message delivery.  
It supports **one-to-one** and **group conversations**, **typing indicators**, **notifications**, and **secure user authentication**.

---

## 🚀 Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Real-Time Communication:** Socket.io  

---

## 🧠 Features

- 🔐 User Authentication (Login / Signup)  
- 💬 Real-time Messaging with Typing Indicators  
- 🧍‍♀️ One-to-One Chat  
- 👥 Group Chat Creation and Management  
- 🔎 Search Users to Start a Chat  
- 🔔 Instant Notifications  
- ➕ Add or Remove Users from Groups  
- 👁️ View User Profiles  

---

## ⚙️ How to Run Locally

1. **Clone the project**  
   Open your terminal and clone the repository.  

2. **Go to the project directory**  
   Navigate to the folder where you cloned the project.  

3. **Install backend dependencies**  
   Run `npm install` inside the main folder.  

4. **Install frontend dependencies**  
   Navigate to the `frontend` folder and run `npm install`.  

5. **Start the backend server**  
   Run `npm run start` inside the main folder.  

6. **Start the frontend client**  
   Inside the `frontend` folder, run `npm start`.  

7. The application will be available on your local machine (usually at **localhost:3000**).
